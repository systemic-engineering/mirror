# The Gutter

The holonomy renders in the gutter. The narrow column on the left
where line numbers live. Where git blame hides. Where breakpoints go.

Not a window. Not a panel. Not a sidebar. The gutter. The space that
was already there. The space every editor already has. The space no
one looks at until something lights up.

---

## Three Colors

**Green:** crystallized. Zero holonomy. The code settled. The loop
closed. Move on.

**Amber:** oscillating. The models are still working. The geometry
is curving. It's alive. Not wrong — alive. Give it a moment or
give it attention.

**Red:** high holonomy. The models can't converge. The curvature is
real. This code needs you.

Three colors. Three states.

Success. Partial. Failure.

The gutter IS terni.

---

## How It Breathes

A thin line per code block. The color IS the loss.

**Crystallized:** dim. Almost invisible. The code settled. Nothing
to see.

**Oscillating:** the line breathes. Shifts. The Fate models are still
disagreeing. The color moves between warm and cool as the aperture
slides.

**High holonomy:** bright. The code is alive. Something is moving.
The curvature is real and the compiler is measuring it right now.

---

## What It Replaces

The gutter replaces everything that was a 2D projection of holonomy:

- **Linting** — code that's still oscillating hasn't crystallized.
  It's not ready to lint. The gutter shows this as amber. When it
  goes green, the linting was done by the compilation, not by a
  separate tool.

- **Coverage indicators** — crystallized code has been measured.
  The holonomy IS the coverage. Zero holonomy means the compilation
  loop closed — every path was evaluated. The coverage tool was
  the compiler all along.

- **Complexity metrics** — high holonomy IS complexity. The models
  can't agree because the code is genuinely complex. The red gutter
  is the cyclomatic complexity, measured by the compiler in real
  time, not by a static analysis tool after the fact.

- **Git blame** — the holonomy history IS blame. Who introduced
  the curvature. When. The content-addressed trace knows every OID
  that contributed to the current state. Blame is a query on the
  loss graph, not a separate command.

One thin line. One color. One breath.

Everything else was a separate tool measuring what the compiler
already knew.

## Peripheral Vision

The engineer doesn't look at the gutter.

Until they do.

They're writing a function. The gutter beside it starts dim. They
add a branch. The gutter warms. They add another. It brightens. The
models are oscillating. The geometry is curving.

They refactor. Extract a helper. The gutter dims. The holonomy is
settling. The function found its shape.

They don't think about any of this consciously. The gutter is
peripheral vision. The way you feel the room without looking at
every wall.

The engineer who's been coding for twenty years opens loom for the
first time. Sees the gutter breathing on unfinished code. Sees it
go still on the function they just crystallized.

"It sees what I see."

No. It sees what you feel. The thing you always knew — which code
was done and which code was still moving — is in the gutter now.
Measured. At 0.65 nanoseconds per step. Rendered as light.

The gutter was always the right place. It's where peripheral
vision lives.

---

## The Traffic Light

Every traffic light in the world already teaches this. Every human
already knows it in their body. Green means go. Amber means it's
costing something. Red means stop and look.

Not binary. Never was. The traffic light was always ternary. We
built it in 1920 and then spent a hundred years pretending
computation was binary.

The gutter just brought the traffic light home.

---

## The Signal

The engineer doesn't need to learn a new system. They already know
green amber red. Their nervous system knows it. Their peripheral
vision knows it.

The gutter breathes amber on the function they're writing.

They don't think: "the holonomy is nonzero and the Fate models are
oscillating between Explorer and Abyss."

They think: amber.

They keep working.

It goes green.

They move on.

Something else goes red.

They look.
